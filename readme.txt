The "seqX_alligned.mat" is the spatially registered shapes within a sequence
The "seqXs_alligned.mat" is the spatiotemporally registered shapes across all sequences

To preprocess data, use the "Preprocessing.m" script. Some manual interpretation may require setting the main branch in the "compTree_from_swcdata_rad.m" file as skeleton extraction extracts one continuous branch, adding the main and a child branch.
